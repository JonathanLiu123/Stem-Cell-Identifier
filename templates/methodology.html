Our Methodology


        We created a machine learning model using Google Brain's Tensorflow software to predict the likelihood that a particular combination of genes will result in a stem cell. In particular, we utilized the Keras deep learning library to create a neural network that takes gene expressions as inputs and creates a regression model. This regression model outputs a probability that the given combination of genes will result in a stem cell. We chose 6 genes known to be linked to stem cells for our model: FGF13, GDF3, SKIL, ERAS, TRIM28, and ZFX. We trained and tested our model using the expression data of these genes in both stem cell lines and non-stem cell lines, which was obtained through StemCellDB, an open-source database with genetic information about stem cells and normal cells.

Accuracy of our Model


        After 10 trials of running our program, we found that the models that our machine learning program created were very accurate, with only an average error of 3.726% probability of being a stem cell. Additionally, these 10 trials correctly categorized every single stem cell and non stem cell in the dataset, with 94.167% of these cells being categorized correctly within the 95% confidence interval. Limitations of our Model


        While our model proved to be very accurate in predicting the likelihood that a certain set of genes corresponds to a stem cell, it does still have some limitations. We currently only use 6 prominent genes in our model, while there are at least 250 genes known to be correlated with stem cells in some way. An extension of this project is to integrate more of these genes into the model for greater accuracy. We welcome others to continue developing and extending our ideas or code.
